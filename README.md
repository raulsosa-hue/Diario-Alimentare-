
# Diario Alimentare & Esercizio (Flutter)

Progetto semplice per registrare:

- pasti principali (colazione, pranzo, cena)
- pasti aggiuntivi / spuntini
- esercizio fisico

e **esportare tutto in un file CSV apribile con Excel**.

## Come usare

1. Installa Flutter sul tuo computer.
2. Clona o copia questo progetto.
3. Da terminale, nella cartella del progetto:

```bash
flutter pub get
flutter run
```

## Esportazione Excel (CSV)

Nella schermata **"Esporta in Excel (CSV)"** premi il pulsante:

- l'app genera un file `.csv`
- il file viene salvato nella cartella Documenti dell'app
- puoi aprirlo con Excel, Numbers, Google Sheets, ecc.
